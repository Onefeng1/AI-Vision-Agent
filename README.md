# 🤖 AI Vision Agent (C# + Halcon)

An AI-powered industrial vision development system that automates defect detection pipeline generation using multi-agent architecture.

---

## 🚀 Overview

This project aims to automate the development of industrial vision inspection systems using AI Agents.

Instead of manually designing image processing workflows, this system can:

- Parse inspection requirements (scratch, hole, stain detection)
- Automatically generate Halcon + C# code
- Optimize parameters (threshold, filters, regions)
- Output defect detection visualization results

---

## 🧠 Architecture

The system is built with a multi-agent pipeline:

### 1. Requirement Agent
Transforms user input into structured inspection tasks.

### 2. Planning Agent
Designs image processing workflows:
- Preprocessing
- Thresholding
- Morphology
- Region analysis

### 3. Code Generation Agent
Generates Halcon + C# code automatically.

### 4. Optimization Agent
Tunes parameters:
- Threshold values
- Kernel sizes
- Region filtering rules

### 5. Validation Agent
Executes pipeline and outputs:
- Defect bounding boxes
- Segmentation maps
- Zoomed defect regions
