
# System Architecture

The AI Vision Agent system is built on a multi-agent architecture designed to automate industrial vision development.

## Workflow

1. Requirement Parsing
2. Task Structuring
3. Pipeline Planning
4. Code Generation
5. Parameter Optimization
6. Result Validation

## Agents

- Requirement Agent
- Planning Agent
- Code Generation Agent
- Optimization Agent
- Validation Agent

## Key Idea

Transform traditional manual vision development into an automated AI-driven pipeline.
